Upon completion of this course, you should be able to
  • Demonstrate understanding and application of web markup language (HTML5) and web style language (CSS3)
  • Publish Web documents to a web server for hosting
  • Create a semantically correct standards-compliant web page
  • Create valid XML data documents
  • Transform XML document to other formats using XSLT (i.e. XML to XML, XML to HTML, etc.)
